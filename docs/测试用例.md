# 测试用例

> **icar-ros2-patrol** 测试用例与测试记录

---

## 测试说明

本文档记录系统各模块的测试用例和测试结果。测试分为：

- **单元测试**：单个节点/函数的功能验证
- **集成测试**：多节点协同通信验证
- **系统测试**：完整巡检任务流程验证
