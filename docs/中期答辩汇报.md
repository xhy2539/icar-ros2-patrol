# iCar 智能巡检小车 — 中期答辩汇报

> 日期：2026.07.11 | 版本：v0.2-mid

## 一、项目概况

基于 Jetson Orin Nano 的 ROS2 智能巡检小车，支持语音对话、自主导航、视觉检测、传感器采集，具备局域网/云平台双模通信。

| 项目 | 信息 |
|------|------|
| 平台 | Jetson Orin Nano (4GB) + ROS2 Foxy |
| 硬件 | 思岚A1激光雷达 / 奥比中光深度相机 / AT32底盘 / 8个IoT传感器模块 |
| 开发方法 | Mock 先行、接口统一、松耦合并行开发 |

## 二、整体架构

```
用户 ──🎤语音/📱APP──▶ task_manager (调度大脑)
                          │
        ┌─────────────────┼─────────────────┐
        ▼                 ▼                 ▼
   导航模块(曹莹)     视觉模块(韦雪)     LLM模块(王璐)
   激光雷达SLAM       YOLO目标检测      语音理解+报告
   自主导航+避障      目标跟踪+录制      模板兜底
        │                 │                 │
        └─────────────────┼─────────────────┘
                          ▼
                   底盘控制 + 传感器
                          │
                    ┌─────┴─────┐
                    │  云平台通信  │
                    │ MQTT+WebSocket│
                    └─────────────┘
```

## 五、技术亮点

### 1. Mock 先行，松耦合并行开发
所有模块通过统一 Topic 接口通信，task_manager 不感知对面是 mock 还是真模块。5 人并行开发，互不阻塞。

### 2. 双模通信
- **局域网**：WebSocket 直连，低延迟视频流 + 控制
- **远程**：MQTT 云平台中转，手机 4G/小车 NBIOT 通信
- 云服务器已部署 24h 运行

### 3. 本地 LLM 语音交互
- llama.cpp-omni 编译完成，MiniCPM-o 4.5 模型(7.3GB)下载
- M5 Metal 加速验证通过，LLM+TTS 生成正常
- 答辩可演示语音对话能力

### 4. 安全机制
- 三级安全：task_manager 紧急停止 > obstacle_avoid 避障 > LLM 不碰底盘
- 障碍物 < 0.3m 自动停止，传感器超标触发告警
