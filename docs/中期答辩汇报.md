# iCar 智能巡检小车 — 中期答辩汇报

> 日期：2026.07.11

## 一、项目概况

基于 Jetson Orin Nano 的 ROS2 智能巡检小车，支持底盘控制、SLAM 建图、视觉检测、APP 控制、云平台远程通信。

| 项目 | 信息 |
|------|------|
| 平台 | Jetson Orin Nano (3.2GB 可用) + ROS2 Foxy |
| 硬件 | 思岚A1激光雷达 / 奥比中光深度相机 / AT32底盘 |
| 开发方法 | Mock 先行、统一 Topic 接口、松耦合并行开发 |

## 三、架构设计

```
用户 ──📱APP──▶ task_manager (调度大脑)
                    │
      ┌─────────────┼─────────────┐
      ▼             ▼             ▼
  navigation    vision         llm_gateway
  lidar/slam    camera/detect  parse/report
  obstacle      tracker        fallback
      │             │             │
      └─────────────┼─────────────┘
                    ▼
              底盘 + 传感器
                    │
              ┌─────┴─────┐
              │  云平台通信  │
              │ MQTT+WS 桥接 │
              └─────────────┘
```
