# 积水分割训练指南

项目使用 YOLO11 实例分割模型识别地面积水。公开数据集包含 5,777 张图片，
按 4,044 / 1,155 / 578 划分为训练、验证和测试集，许可证为 CC BY 4.0。

数据集不进入 Git 历史，通过 GitHub Release 分发：

<https://github.com/xhy2539/icar-ros2-patrol/releases/tag/water-dataset-v1>

## 1. 安装依赖

```bash
python3 -m pip install ultralytics
```

## 2. 下载数据

```bash
cd ~/icar-ros2-patrol
bash scripts/download_water_dataset.sh
```

脚本会下载并解压数据到 `datasets/water_public`，同时修正 Roboflow 导出的
数据路径和类别名。

## 3. 训练

```bash
python3 scripts/train_water_seg.py
```

默认参数为 50 轮、640 输入尺寸和 batch 4。NVIDIA GPU 会自动使用 CUDA，
Apple Silicon 会自动使用 MPS。可以按显存调整：

```bash
python3 scripts/train_water_seg.py --batch 8 --device 0 --name water_seg_gpu_v1
```

最佳权重位于：

```text
runs/water/<name>/weights/best.pt
```

## 4. 测试

```bash
yolo segment predict \
  model=runs/water/water_seg_v1/weights/best.pt \
  source=datasets/water_public/test/images \
  conf=0.25 \
  save=True
```

预测结果默认保存在 `runs/segment/predict`。正式接入小车前，还应使用小车摄像头
拍摄的积水、地面反光、阴影和地垫图片进行额外测试。
