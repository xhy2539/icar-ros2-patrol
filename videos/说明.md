# 演示视频存放说明

此目录用于存放项目演示视频。

## 命名规范

```
{日期}_{内容}.mp4
```

示例：
- `20260707_底盘控制演示.mp4`
- `20260714_SLAM建图演示.mp4`
- `20260721_自主导航演示.mp4`
- `20260728_目标检测演示.mp4`
- `20260804_完整巡检演示.mp4`
