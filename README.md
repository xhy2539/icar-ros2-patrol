# 基于ROS2的智能巡检小车系统

> **icar-ros2-patrol** — 基于 ROS2 的智能巡检小车实训项目

---

## 项目简介

本项目基于 ROS2 (Robot Operating System 2) 构建一套智能巡检小车系统，实现小车运动控制、雷达避障、SLAM 建图、自主导航、视觉检测、多传感器数据采集、任务日志展示等功能。项目以模块化方式分工协作，最终可扩展接入 LLM（大语言模型）实现智能任务解析与巡检报告生成。

## 功能优先级

| 优先级 | 功能模块 | 说明 |
|--------|----------|------|
| **P0** | 小车底盘运动控制 | APP 远程控制小车前进/后退/转向/停止 |
| **P0** | ROS2 基础通信 | 节点间 Topic/Service/Action 通信框架 |
| **P0** | 激光雷达数据采集与避障 | 雷达扫描 + 障碍物检测 + 紧急停车 |
| **P1** | SLAM 建图 | 实时建图并保存地图 |
| **P1** | 自主导航 | 目标点设置 + 路径规划 + 自动行驶 |
| **P1** | 视觉目标检测 | OpenCV/YOLO 实现目标识别 |
| **P1** | 传感器数据采集 | 温湿度/烟雾/PM2.5/光照/气压采集展示 |
| **P2** | 任务日志与巡检报告 | 任务状态记录 + 巡检数据汇总 |
| **P2（加分项）** | LLM 任务解析 | 自然语言指令理解与巡检报告自动生成 |

## 快速启动

> 以下为占位说明，完整启动流程将在各模块开发完成后补充。

```bash
# 1. 确保 ROS2 环境已 source
source /opt/ros/humble/setup.bash

# 2. 构建工作空间
colcon build --symlink-install

# 3. source 工作空间
source install/setup.bash

# 4. 启动完整演示
./scripts/start_demo.sh
```
